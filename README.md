# GPX to Instagram

Tiny utility to convert GPX track from Strava to an Instagram image.

Made just for fun. 

Usage
```shell
gpx-to-instagram -g your-ride.gpx -i input-image.jpeg -o output-image.jpg -a K1ZMT
```
